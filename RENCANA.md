# Rencana Belajar Database

## Alur Belajar

1. **01_Konsep_Dasar** — SQL dasar dulu (CREATE, SELECT, INSERT, JOIN). Paham relational model.
2. **02_Storage_Indexing** — Paham gimana database nyimpen data di disk. Sequential vs Index.
3. **03_Query_Execution** — Paham gimana database ngejalanin query lo. EXPLAIN.
4. **04_Transactions** — Paham ACID, transaction, locking. Biar data gak kacau pas banyak user.
5. **05_Normalization** — Paham cara desain tabel yang bener. 1NF → 2NF → 3NF.

## Tips

- Jangan skip urutan. Topik 1-2 itu fondasi.
- Jalanin semua SQL manual di terminal biar nempel.
- Abis SQL, baru bandingin kode di 4 bahasa (JS, Go, Java, C).
- Kalau mentok, tinggal tanya YongMa.
